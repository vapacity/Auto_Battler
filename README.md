# Auto_Battler
同济大学2022级软件工程程序设计范式期末项目

##1.添加了小小英雄的受击和攻击效果
受击：变红同时后撤一步  
攻击：射出火球，，这里素材可以更改

## 2.封装了鼠标事件监视器
直接将LittleHero加到场景中即可实现右键点击


##3.添加了小小英雄的敌我判断
暂时设定 具体得根据后续的player等进行更改  
具体在create函数中添加了一个变量1为敌人0为自己  
其中只有自己的小小英雄能实现移动  
	littleHero = LittleHero::create(filename,0 or 1);
